%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
 VECT_ARROW(P0, P1, COLOR) 

 VECT_ARROW(P0, P1) plots a line vector with arrow pointing from point P0
   to point P1. 

   Example:
       %3D vector
       p0 = [1 2 3];   % First point p0
       p1 = [4 5 6];   % Second point p1
       vectarrow(p0, p1)

 See also DRAW_LINK, DRAW_PATCH, DRAWROBOT3D.
